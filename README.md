
# React Performance Optimization Demo

## Features Implemented:
1. React.memo (Item component)
2. useMemo (list generation)
3. useCallback (event handler)
4. Lazy Loading (React.lazy + Suspense)
5. Basic list optimization (simulate virtualization)
6. Ready for Lighthouse & DevTools profiling

## Run Project:
npm install
npm run dev

## Analyze Performance:
- Use React DevTools Profiler
- Use Lighthouse in Chrome DevTools
